# Project 06: Microsoft Entra ID & Hybrid Identity Management

## Overview
This project demonstrates the administration and security of a hybrid identity environment using Microsoft Entra ID (formerly Azure AD). Building upon an on-premises Windows Server Active Directory infrastructure, this lab integrates cloud-native identities, B2B collaboration, and role-based access control (RBAC) to simulate a modern enterprise identity perimeter for Stark Enterprise.

## Core Technologies
* Microsoft Entra ID
* Microsoft Entra Connect (Hybrid Sync)
* Windows Server Active Directory
* PowerShell (Microsoft.Graph module)

## Project Phases
* **Phase 01: Hybrid Identity:** Validated directory synchronization and Source of Authority for on-premises synced accounts.
* **Phase 02: Cloud Users & B2B Collaboration:** Provisioned cloud-native employee accounts and securely invited external contractor identities using B2B Guest features.
